Stylesheets

Stylesheets are your css files, which are going to be processed. As in the pages, you can use partials and helpers inside your stylesheets, for more flexible generation. For instance, it can be used for passing build variables or pices of html to your css via ejs:

// Button Component:
// <%- partial('button', { label: 'Ok'}) %>
.button {
  color: <%- buttonColor %>
}

Which can result in:

// Button Component:
// <button class="btn">Ok</button>
.button {
  color: RebeccaPurple;
}

Pages go into source/stylesheets directory.
